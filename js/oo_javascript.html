Object Oriented JavaScript
A set of arguments I've created for proving JavaScript is now an Object Oriented Language.

Has JavaScript developed into an Object Oriented Language?

I believe it has. One can create objects using the new operator. There is a built in set of predefined objects but JavaScript extends even farther to incorporate the handling of user created objects as well.

There are both private and public functions/variables in relation to user defined JavaScript objects. After reviewing the below documents you will see how JavaScript handles private and public data.

An intermediate level knowledge of JavaScript/HTML is required to understand the below argument. This is not a tutorial although it may teach intermediate JavaScript/HTML programmers new tips or tricks.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
<html>
	<head>
		<meta http-equiv="Content-Type" content="text/html; charset=iso-8859-1">
		<title>Object Oriented JavaScript</title>
		<script type="text/javascript" src="example.js"></script>
	</head>
	<body>
		<script type="text/javascript">
			var myObj=new cost();
			document.writeln("Example Version: "+myObj.version+"<br>");
			
			//attempt to set the private var using a private function
			try 
			{
				myObj.saveData(10);
			}
			catch(e)
			{
				document.writeln('<span style="color:red">'+e+'</span><br>')
			}
			//attempt to access the private var directly
			document.writeln("Price is: $"+myObj.price+"<br>");
			
			//access the private function to set the private var using a public function
			myObj.enterData(10);
			//access the private var using a public function
			document.writeln("Price is: $"+myObj.getPrice()+"<br>");
			//access the public array
			for(var i=0;i<myObj.count.length;i++)
				document.write(myObj.count[i]+', ');
			document.writeln('<br>');
		</script>
	</body>
</html>

JavaScript Document
Copy and paste this code into example.js so it can be used and make sure it is in the same folder as example.html.

//JS constructor (which is kinda viewed as a class)
function cost()
{
	//private variables
	var price=0;
	
	//public variables
	this.version=0.1;
	this.count=new Array(); //Built in predefined object Array()
	this.count[0]=1;
	this.count[1]=2;
	this.count[2]=3;
	
	//private functions
	function saveData(a)
	{
		price=a;
	};
	
	//public funcions
	this.enterData=function(a){
		saveData(a);
	};
	this.getPrice=function(){
		return price;
	};
}
