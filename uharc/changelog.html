UHARC CMD Version 2.1.1.2 released 08/11/2009

2.1.1.2
Updated Feature:
	1. Changed settings defaults.  compress.pkd is default
	   file name.  Removed excluding .pkd and .uha files
	   by default.
	2. Encryption command now uses the same settings which
	   are set by the user in settings.exe
	3. Updated settings to write encryption command.
	4. Updated the NSIS core of UHARC CMD to NSIS 2.45
Changed Feature:
	1. Added enc_cmd.bat to installation files so the
	   encryption process uses the same settings which
	   are set by the user in settings.exe

To see a full changelog review the changelog of the installed package.
